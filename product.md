# Product Notes

## Business Context
- Promotional site for "The Internet Yami-ichi Arnhem 2025" event, showcasing branding and participant information.
- Emphasizes playful, retro-internet aesthetic aligning with event theme (net art, creative technology).
- Supports both informational needs (venue, schedule) and experiential flair (interactive chaos mode, animated hands).

## Existing Features
- **Event overview**: Title, venue address, date (with auto-generated weekday), and time.
- **Brand assets**: Primary and alternate logos displayed prominently with fallback treatment if images fail.
- **Artist roster**: Individual cards with bios and outbound links (Instagram, personal site placeholders).
- **Interactive chaos mode**: Toggle button switches layout to draggable, randomly offset cards for playful user engagement.
- **Animated hand overlay**: Decorative hands that orient toward cursor, adding motion and depth.
- **Accessibility touches**: Visually hidden headings for screen readers, focus states on interactive elements, reduced-motion support.

## User Workflow
1. **Arrival**: User lands on the single-page site, immediately sees header branding and ambient hand animations.
2. **Event details**: Scroll or glance to venue section for address, date, and time; weekday label ensures clarity for planning.
3. **Explore artists**: Review artist cards; optional actions to visit social media or websites via icon buttons.
4. **Engage playfully**: Users can press "I HATE HTML" button to enter chaos mode, drag cards around, then toggle back.
5. **Closure**: Footer reinforces branding and ownership; user leaves with event information and impression of creative tone.
